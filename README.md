# ‚ú® Features

Window Manager: Hyprland.

Bar: Waybar (Custom Modules, Neon borders & Glow effects).

Launcher: Rofi (Custom Theme "Neon Tech").

Terminal: Kitty + zsh (JetBrainsMono Nerd Font).

Notifications: SwayNC (Control Center).

Logout Menu: Wlogout.

# üöÄ Installation

## Prerequisites

- OS: Arch Linux.

- Git installed (sudo pacman -S git).

## Quick Start

Clone the repository:
```
git clone https://github.com/theoneandonly-amiko/Arch-Novaland.git
```
Enter the directory:

```
cd Arch-Novaland
```
Run the installer:
```
chmod +x install.sh
./install.sh
```

# ‚å®Ô∏è Keybinds (Cheatsheet)

| Key Combination | Action |
| :--- | :--- |
| Super + Q | Open Terminal (Kitty) |
| Super + E | Open File Manager (Thunar) |
| Super + R | Open App Launcher (Rofi) |
| Super + W | Wallpaper Switcher (Gallery Mode) |
| Super + G | Toggle Gamemode (Disable animations/blur for performance) |
| Super + L | Lock Screen |
| Super + M | Power Menu (Logout/Reboot/Shutdown) |
| Super + V | Toggle Floating Window |
| Super + K | Show Keybinds Hint (some may missing! Forgive me xd) |
| Alt + Tab | Switch Workspace |
| Super + Print Screen | Screenshot (Full screen -> Pictures) |
| Print Screen | Screenshot (Region -> Clipboard) |
